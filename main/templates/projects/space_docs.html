Space Invaders Documentation

This Space Invaders game is a faithful remake of the classic arcade shooter, built using Pygame. It features smooth player controls, multiple enemy waves, dynamic enemy movement, and classic sound effects for an authentic retro experience.

Classic Enemy Waves

Player Shooting

Shoot bullets with cooldown and laser sound effects

Player Lives & Game Over

Three lives with game over screen and restart option

Collision & Scoring

Bullet-enemy collision triggers explosions, enemy death, and score increment

Game Mechanics

The player controls a spaceship moving horizontally along the bottom of the screen, firing bullets to eliminate incoming alien enemies. Enemies move as a group, shifting direction when hitting screen edges and descending closer to the player over time.

When an enemy is hit, it triggers a falling animation line and explosion sound, adding visual feedback. The player has three lives and loses one if an enemy collides with the ship or its falling line touches the player. The game restarts once all lives are lost.

Technical Implementation

The game utilizes the following systems:

  • Sprite rendering for player, enemies, bullets, and UI
  • Enemy group movement with direction reversal and descent on edge collision
  • Bullet management with cooldown timer and collision detection
  • Audio with looping background music and event sounds (laser, explosion, game over)
  • Simple UI with start menu, play/quit buttons, score, and lives display

The game loop manages event handling, state transitions (menu, playing, game over), and rendering at a consistent 60 FPS for smooth gameplay.
